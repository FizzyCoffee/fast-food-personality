## What is this

A 12-question personality quiz that categorizes you as one of 8 fast food items. It is the most accurate personality test ever devised\*\*.

### The 8 personality types

| | | |
|---|---|---|
| 🍔 **Hamburger** — The Reliable Classic | 🌭 **Hot Dog** — The Unhinged Minimalist | 🍜 **Ramen** — The Deep Thinker |
| 🍕 **Pizza** — The Universal Crowd-Pleaser | 🌮 **Taco** — The Beautiful Disaster | 🍣 **Sushi** — The Refined Perfectionist |
| 🍗 **Fried Chicken** — The Charmer With Secrets | 🍩 **Donut** — The Sweet Optimist | |

## Features

- 🎉 Confetti cannons (two of them, because one was deemed insufficient)
- ✨ Ripple effects on every button (we take clicking very seriously here)
- 🎰 Slot-machine emoji reveal (your fate takes ~1 second to decide)
- 🍟 Food-emoji particle bursts (real food cannot be simulated; pixels must suffice)
- 📜 Clickable menu — browse all 8 types and judge other people
- 💾 Share button (for bragging)

## Running it locally

It's three static files. Open `index.html`. That's it. You did it.

If you need this to be more complicated, you can also run:

```
python3 -m http.server 8000
```

and navigate to `localhost:8000`. But honestly, double-clicking the file is fine.
